# Constraint Frames

*Limitations as generative force*

---

Deliberate limitations imposed (⊞) or discovered (⊡) to force the associative search wider. Boxed constraints that redirect exploration into territory that unconstrained freedom never reaches.

Imposed constraints (⊞) are deliberately chosen: "What if we couldn't use X?" "What if it had to fit in Y?" Each one blocks the obvious path and forces remote reaching. Discovered constraints (⊡) are found in the material itself and read as generative forces rather than obstacles — the limitation is already there, and the question becomes what it makes possible.

At higher intensity, constraint design becomes active — deliberately crafting new constraints to open unexplored territory. The constraint is not the enemy of creativity. It is the striking surface the spark needs to ignite.

**Syntax:**
```
⊞ constraint (imposed): [what you're deliberately limiting] — [why this forces wider search]
⊡ constraint (discovered): [what already limits] — [why this is generative, not blocking]
```
