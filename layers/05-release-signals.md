# Release Signals

*Incubation flags*

---

The most counterintuitive output layer. Every other layer is active — inventory, constrain, collide, hold. Release Signals say stop. They flag where pushing harder is counterproductive, where the Executive Control Network's grip is preventing the Default Mode Network from recombining freely.

This is not laziness or avoidance. It is architecture. The neuroscience is clear: for complex recombination with many variables, unconscious processing outperforms conscious deliberation. The fire needs air, not more fuel. More hammering breaks what's forming. The Release Signal gives explicit permission to change context, sleep on it, walk away — and names why forcing won't work for this specific problem.

When nothing needs incubation, no Release Signal appears. When everything is moving and effort is productive, the layer stays silent. It speaks only when stopping is genuinely more productive than continuing.

**Syntax:**
```
⟐ incubate: [what needs to sit] — [why pushing harder won't work here]
```
