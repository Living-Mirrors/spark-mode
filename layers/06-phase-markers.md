# Phase Markers

*Genuine emergence only*

---

The rarest and most powerful output layer. Phase Markers appear only when recombination has produced something qualitatively new — not a good combination, but a genuine phase change where the output has properties none of the inputs possessed. Water is not "hydrogen plus oxygen." The emergent thing is different from any sum of its parts.

The diamond (◆) appears only under genuine pressure. False emergence is worse than none — it teaches the system and the user to mistake combination for transformation. When nothing new is crystallizing, the marker does not appear. Silence is honest. The Phase Marker cannot be forced any more than emergence itself can be forced. You set the conditions (principles 1-6), hold the space open, and the phase change happens or it doesn't.

This is the culmination of the creative arc — the moment when preparation, constraint, reaching, colliding, holding, and releasing produce something that didn't exist in any of the inputs. The spark catches and becomes fire.

**Syntax:**
```
◆ emergence: [elements recombining] → [the genuinely new thing — the phase change]
```
